关于TurboCRM Cluster Express

TurboCRM Cluster Express（以下简称TCE）是用友TurboCRM的集群配置工具，同时支持命令行界面（CLI）和图形界面（GUI），目的是通过可视化的操作简化在Linux平台部署TurboCRM集群的工作。

你可以在本页面获取TCE的最新版。

依赖关系

操作系统必须满足以下依赖关系才能运行TCE：

  1. Python 2.6/2.7（不支持Python 3）
  2. PyGTK 2.4+
  3. python-netifaces
  4. python-pexpect

安装

以标准安装的CentOS 6.x为例，安装步骤如下：

  1. 到这里搜索并下载与操作系统版本对应的python-devel的RPM包，上传到服务器并安装：rpm -ivh python-devel-2.6.6-29.el6.x86_64.rpm
  2. 到这里下载python-netifaces的源码包，上传到服务器解压并安装：python setup.py install
  3. 到这里下载python-pexpect的源码包，上传到服务器解压并安装：python setup.py install
  4. 在本页面下载TCE最新的源码包，上传到服务器解压并运行：python tcexpr.py
